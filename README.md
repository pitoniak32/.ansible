# How to use
1. install git 
  - MacOs: `xcode-select --install`
  - Manjaro: `pacman -S git`
  - Ubuntu: `sudo apt install git`
2. clone this repo `git clone git@github.com:pitoniak32/.ansible.git`
3. run `./bootstrap.sh`
